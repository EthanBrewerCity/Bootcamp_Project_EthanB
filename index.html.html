Bootcamp 2021: Bouncing Balls With Gravity

In my project during bootcamp, I have built a program which creates 2 bouncing balls which are affected by gravity and a dampening effect upon colliding with any walls or the other ball itself. In addition, the program shows the elapsed time that the program has been active, how many times each ball has collided with any wall and how many times the two balls have collided with eachother. This program uses PVectors for the positions, the speed and the gravity affecting the balls, it also uses if statements to register if the walls have been reached by the balls, and keyPressed to change the background of the program.
